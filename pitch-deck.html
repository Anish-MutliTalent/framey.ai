The moat

Four layers stand between “AI” and “professional.”

LAYER 1

Native action

Every agent works through After Effects' own scripting & UI pathways — exactly as a human would. No screen-reading hacks.

LAYER 2

Deterministic replay

Every action is logged to framey-action-log.jsonl and re-runnable against a fresh driver. Never a lucky one-off.

LAYER 3

Automated QC

Before finalizing, Framey validates timing, alignment, and render integrity against professional standards — and fixes what it finds.

LAYER 4

You

The project file stays fully open. A human can review, tweak, or override any decision the agents made.

Quality layer 3

It checks its own work before you see it.

CHECKS RUN

Professional standards, enforced

· Duration sane (≤ 600s)
· At least one layer present
· Timing integrity — inPoint < outPoint, within comp
· Text on-canvas — not off the edges
· Keyframes valid — ≥ 2 per animated property, inside comp
· Render queue non-empty, real output path
· Project saved to disk

QC REPORT

Runs on the real thing

It drives the actual editor studios use.

MODE 1 · DEFAULT

Simulator

In-process comp model. No After Effects install needed. Ships with the app, so anyone can run the full agent loop live and watch the timeline build in real time.

MODE 2 · REAL AE

Real After Effects

Load ae/driver.jsx into a running AE. It connects over TCP, runs each command in an undo group (every action undoable), and pushes the full comp state back to the UI.

$ npm install $ npm run dev # launch the Electron app $ # add a Fireworks API key in Settings, type a prompt, watch the agents build a real comp
